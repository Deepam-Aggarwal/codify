AIM:Write a Program to check whether a number is prime or not.
ALGORITHM:
    Step 1: Start
    Step 2: Enter the number to be checked.
    Step 3: if the number is divisible by any other number except itself or 1 then it is prime i.e. 
            if (number% i==0) the number is not prime.
    Step 4: Print the answer.
    Step 5: Stop
            
CODE:
    #include <iostream>
    int main()
    { 
        int num;
        cout<<"Enter the number:";

        int flag = 0;

        {
            if(num%i == 0)
            {

            }
